# HeoLab Video Content Strategy

**Goal:** Drive traffic to heolab.dev via "Challenge + Showcase" video content across 5 platforms from a single recording workflow.

**Format:** English-only. Short-form (30-60s) for viral reach + Long-form (5-8 min) for SEO. No face cam required — screen record + text overlay + music.

---

## Platforms

| Platform | Format | Priority |
|----------|--------|----------|
| YouTube | Long-form + Shorts | Primary |
| TikTok | Repost Short | High |
| Instagram Reels | Repost Short | Medium |
| Facebook Reels | Repost Short | Medium |
| YouTube Shorts | Auto-clip from Long | Auto |

**Cross-posting workflow:**
```
Record → Edit (CapCut) → Upload YouTube Long → Repurpose.io → TikTok + Instagram + Facebook Reels + YouTube Shorts
```

**Tools:**
- [Repurpose.io](https://repurpose.io) — auto-distribute to all platforms
- [Buffer](https://buffer.com) — scheduling (free plan)

---

## Content Format

### Short-form (30-60s)
```
0-3s   → Hook text + gameplay starts
3-25s  → Gameplay tension (near miss, close call)
25-55s → Win/lose moment + reaction text overlay
55-60s → CTA: "Play free at heolab.dev 🔗 link in bio"
```

**Hook examples:**
- "Can I beat Minesweeper Hard without dying? 👀"
- "Getting 1000 points in Neon Serpent 🐍"
- "This Tetris event just saved my game 💣"
- "Nobody can pass wave 10 in Astro Raid... right? 👾"

### Long-form (5-8 min)
```
0:00  Hook — state the challenge
0:30  Show heolab.dev, navigate to game
1:00  Attempt 1 — fail or near miss
2:30  Attempt 2 — explain tips with text overlay
4:30  Final attempt — win or best score
6:00  CTA — "Play it yourself at heolab.dev" + end screen
```

## SEO: Titles & Descriptions

### Long-form title formula
```
[Challenge] + [Game Name] + [Keyword]
```
Examples:
- "I Tried Beating Minesweeper Hard Mode in Under 60 Seconds"
- "Can You Actually Win at Free Snake Game Online? (Neon Serpent)"
- "Free Tetris Browser Game With RANDOM CHAOS Events — Block Storm"
- "Unlocking All 3 Characters in Pixel Dash Endless Runner"
- "Surviving 10 Waves in Free Space Invaders Browser Game — Astro Raid"

### Shorts title formula
```
[Hook question or statement] + emoji
```
- "Can I survive wave 10? 👾"
- "This power-up just broke the game 💣"
- "New high score unlocked 🔥"

### Description template (Long-form)
```
Play [Game Name] for free at heolab.dev — no download, no account needed.

[1-2 sentence game description]

🎮 Play now: https://heolab.dev/games/[slug]
🕹️ All games: https://heolab.dev/games

#freegames #browsergames #[gamename] #heolab #indiegame #pixelgame
```

### Fixed tags (every video)
`free browser game, play online free, no download game, heolab, indie game, pixel game, browser game 2025`
