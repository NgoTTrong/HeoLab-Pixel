# HeoLab Favicon Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the default Next.js favicon with a custom pixel-art joystick icon in neon green, generated via `ImageResponse`.

**Architecture:** Two files using Next.js file-convention icons — `src/app/icon.tsx` (32×32, browser tab) and `src/app/apple-icon.tsx` (180×180, Apple touch icon). Both use `ImageResponse` from `next/og` with the same pixel-art joystick design scaled proportionally. No external tools needed. The existing `src/app/favicon.ico` is deleted — Next.js 16 serves `icon.tsx` with higher priority for modern browsers anyway.

**Tech Stack:** Next.js 16 App Router, `next/og` ImageResponse, TypeScript

**Design — pixel art joystick (elements as CSS rects, neon green `#39ff14` on `#0a0a0a`):**
- Joystick top: wide square, centered horizontally, upper third of canvas
- Joystick stick: narrow tall rect, below top, connects to base
- Base plate: wide rect, bottom third of canvas

All sizes expressed as fractions of canvas size so the same logic scales to both 32×32 and 180×180.

### Task 1: Create `src/app/icon.tsx` (browser tab favicon, 32×32)

**Files:**
- Delete: `src/app/favicon.ico`
- Create: `src/app/icon.tsx`

**Step 1: Delete the default favicon.ico**

```bash
rm "E:\Personal\GameStation\src\app\favicon.ico"
```

**Step 2: Create `src/app/icon.tsx`**

```tsx
import { ImageResponse } from "next/og";

export const size = { width: 32, height: 32 };
export const contentType = "image/png";

const BG    = "#0a0a0a";
const GREEN = "#39ff14";

export default function Icon() {
  const S = 32; // canvas size

  return new ImageResponse(
    (
      <div
        style={{
          background: BG,
          width: "100%",
          height: "100%",
          display: "flex",
          alignItems: "center",
          justifyContent: "center",
          fontFamily: "sans-serif",
        }}
      >
        {/* Joystick top */}
        <div
          style={{
            position: "absolute",
            background: GREEN,
            width: Math.round(S * 0.375),   // 12px
            height: Math.round(S * 0.25),    // 8px
            top: Math.round(S * 0.125),      // 4px
            left: Math.round(S * 0.3125),    // 10px
          }}
        />
        {/* Joystick stick */}
        <div
          style={{
            position: "absolute",
            background: GREEN,
            width: Math.round(S * 0.125),    // 4px
            height: Math.round(S * 0.25),    // 8px
            top: Math.round(S * 0.375),      // 12px
            left: Math.round(S * 0.4375),    // 14px
          }}
        />
        {/* Base plate */}
        <div
          style={{
            position: "absolute",
            background: GREEN,
            width: Math.round(S * 0.625),    // 20px
            height: Math.round(S * 0.1875),  // 6px
            top: Math.round(S * 0.625),      // 20px
            left: Math.round(S * 0.1875),    // 6px
          }}
        />
      </div>
    ),
    { ...size }
  );
}
```

**Step 3: TypeScript check**

```bash
cd E:\Personal\GameStation && npx tsc --noEmit
```

Expected: no errors.

**Step 4: Verify in browser**

Open `http://localhost:3000` — browser tab should show the new pixel joystick icon instead of the Next.js default.

Also open `http://localhost:3000/icon` to preview the raw PNG.

**Step 5: Commit**

```bash
git add src/app/icon.tsx && git rm src/app/favicon.ico
git commit -m "feat: add pixel-art joystick favicon via icon.tsx, remove default favicon.ico"
```
